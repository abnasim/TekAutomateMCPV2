# Probe Compensation Check

Use this when probe compensation, probe integrity, or passive-probe setup is suspected.

## Procedure

1. Confirm the selected probe attenuation matches the physical probe switch.
2. Run the probe compensation procedure for the target channel if the scope and probe support it.
3. Inspect the compensation waveform and adjust the probe if the square wave is under- or over-compensated.
4. Re-check vertical scale and offset after compensation.
5. Repeat any critical edge or amplitude measurements after compensation completes.

## Notes

- Compensation issues can look like ringing, overshoot, or bandwidth problems.
- Fix compensation before spending time tuning trigger or measurement settings.
